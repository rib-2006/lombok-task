# Lombok Task

This project demonstrates the use of the Lombok library in Java.

## Features
- Use of Lombok annotations (@Data, @AllArgsConstructor)
- Model class without boilerplate code
- Comparison of two objects using equals()
- Verification of hashCode() behavior

## Description
Two Car objects are created with identical values.
Even though they are different objects in memory, the equals() method returns true and hashCode() returns the same value because all fields are identical.

## Technologies
- Java
- Lombok
- Maven
